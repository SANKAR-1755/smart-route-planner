# Smart Route Planner

## 1. Introduction

The Smart Route Planner is a system designed to calculate the shortest path between multiple cities using Dijkstra’s Algorithm. It allows users to input a graph in the form of a distance matrix and computes the optimal route efficiently.

---

## 2. Objective

The objective of this project is to demonstrate how graph-based problems can be solved using algorithms taught in class, specifically a Greedy algorithm.

---

## 3. Algorithm Used

This project uses **Dijkstra’s Algorithm**, which is a **Greedy algorithm**.

* It selects the nearest unvisited node at each step
* Updates shortest distances dynamically
* Ensures the shortest path from source to all nodes

---

## 4. System Design

### Input:

* Number of cities
* Distance matrix (user input)

### Process:

* Apply Dijkstra’s Algorithm on the graph

### Output:

* Shortest distance from source to all cities
* Path taken for each route

---

## 5. Technologies Used

* HTML, CSS (User Interface)
* JavaScript (Frontend logic)
* Node.js with Express (Backend server)

---

## 6. Features

* Interactive GUI for graph input
* Dynamic distance matrix generation
* Displays shortest paths and distances
* Visual representation of routes
